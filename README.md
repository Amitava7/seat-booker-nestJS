## Description

A High-Concurrency Ticket Booking System powered by NestJS and PostgreSQL

## Project setup and run

```bash
$ cp .env.example .env
$ npm install
$ npm run docker:up
$ npm run start
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
